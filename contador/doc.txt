
//Selecionando elementos do meu HTML

1) irá retornar o elemento html pelo seu id

document.getElementById('id do elemento aqui')

2) irá retornar um HTML collection com todos os elementos que possuem a class especificada

document.getElementsByClassName('nome da classe aqui')

3) retorna uma coleção com todos os elementos que são da tag especificada;

document.getElementsByTagName('div')

4) retorna o primeiro elemento do selector que foi passado;
  -- retorna primeira div encontrada
document.querySelector('div')
  -- retorna o primeiro elemento com a class box
document.querySelector('.box')
  -- retorna o primeiro elemento com o id caixa
document.querySelector('#caixa')

é possível ir mais afundo na hierarquia com o query selector
  -- retorna o primeiro elemento com o id caixa que estiver dentro do primeiro elemento encontrado com a class container
document.querySelector('.container #caixa')


5) Igual ao querySelector, porém retorna uma lista de elementos.

  -- retorna todas as tags div que estiverem dentro de um container com 
document.querySelectorAll('#container-div div')
